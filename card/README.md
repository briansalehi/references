# Resource name

- **Type**: (select only one) book, website, course, video, channel, mailing list, manual, slides
- **Pattern**: (select only one) page, chapter, session, episode, playlist, post
- **Presenters**: comma separated list of authors or the team who contributed in creation of this resource
- **Provider**: the company, group, or the platform providing this resource, in case of YouTube it should be the channel name
- **Link**: a link refering to the front page of resource manual or the store where the resource can be purchased from
- **End of life**: an estimated date in form of yyyy when this resource will no longer be useful and can be considered obsolete regardless of how many new editions are published, 0 for never
- **Subject**: the subject that this resource is covering, for example JavaScript
